## 📖 О проекте

По данным IDF Diabetes Atlas 2021, **537 миллионов человек** в мире живут с диабетом - и большинство из них не имеют удобного, безопасного инструмента для самоконтроля. Бумажные дневники теряются, облачные приложения хранят медицинские данные без достаточного шифрования, а существующие решения не связывают питание, физическую активность и уровень глюкозы в единую картину.

**ГлюкоПульс** - это Android-приложение для ежедневного мониторинга глюкозы, разработанное с акцентом на **локальную безопасность** и **простоту ввода**. В данной версии приложения все данные хранятся исключительно на устройстве, зашифрованы AES-256 и защищены биометрической аутентификацией.

## ✨ Ключевые возможности

### 📊 Аналитика и мониторинг
- Добавление показания в **2 тапа** с тегами: стресс, спорт, гипо, гипер, болезнь, алкоголь
- Интерактивные графики на `fl_chart` с фильтрами **День / Неделя / Месяц / Всё**
- **Расчёт HbA1c** по формуле Nathan 2008: `HbA1c = (Avg + 46.7) / 28.7` — без похода в лабораторию
- Настраиваемые целевые диапазоны мин/макс под каждого пользователя

### 🍽️ Дневник питания
- Учёт продуктов с БЖУ и калориями
- Связь приёмов пищи с показаниями глюкозы

### 🔒 Безопасность (Privacy First)
- Локальное хранилище **Hive** с шифрованием **AES-256** — никакой передачи данных на сервер
- **PIN + биометрия** через `local_auth ^3.0.1` (отпечаток, лицо, PIN устройства)
- Скрытые уведомления (`NotificationVisibility.private`) — диагноз не виден посторонним на экране блокировки

### 📱 Удобство и интеграция
- **Home Screen Widget** - актуальный уровень сахара прямо на рабочем столе
- **Умные напоминания** о замерах через `flutter_local_notifications`
- **Экспорт в CSV** через системный share sheet — для передачи врачу или личного архива
- Три темы: 🌙 Dark / ☀️ Light / ⚙️ System — переключение без перезапуска

## 📲 Установка

### Через APK (рекомендуется)

1. Скачайте последний APK из раздела [Releases](https://github.com/Mdcrych/GlukoPulse/releases/latest)
2. Разрешите установку из неизвестных источников в настройках Android
3. Установите APK и запустите приложение

### Сборка из исходников

**Требования:** Flutter SDK ≥ 3.6.0, Android Studio / VS Code с Flutter-плагином, Android SDK (API 21+)

```bash
git clone https://github.com/Mdcrych/GlukoPulse.git
cd GlukoPulse
flutter pub get
flutter run                        # отладка
flutter build apk --release        # сборка релиза
```
